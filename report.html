DSSP + ASA/SASA：可做的下游分析清单

结构特征来自 DSSP (secondary structure + ASA) 与 FreeSASA (SASA total/main/side)。 下游分析思路对齐 Cell 2023 文章（Geffen et al., S009286742300781X）的“PTM 分层解释 / 串扰 / 3D clumps / 治疗可行性”框架。

Secondary structure (DSSP SS) Solvent accessibility (ASA/SASA) Enrichment / Statistics 1D neighbourhood (±5 aa) 3D proximity / Clumps Variant overlay (SNP/ClinVar/TCGA) Actionability / Therapeutics

Practical notes（建议写进 Methods 的关键点）

1) Exposure definition

你目前拿到的是 ASA/SASA（绝对面积）。跨氨基酸类型比较建议转成 RSA = ASA / MaxASA 或使用分位数阈值（e.g., top 25% = exposed）。

2) Secondary structure bins

DSSP 可先合并为 H (helix) / E (beta) / C (coil/loop)， 便于做富集与解释；也可保留原始 DSSP 8-state 作为更细粒度特征。

3) 1D vs 3D crosstalk

先做 ±5 aa 的线性邻域共现（协作/竞争），再用 3D 距离（e.g., Cα < 8Å）做 clumps，可显著提高“机制可信度”。

4) Output design

建议输出 4 张核心表：SS/Exposure enrichment、 1D crosstalk pairs、3D clumps、 Variant→PTM loss candidates。

How to report in a paper（建议的 Results 组织方式）
Figure set A: Structural preference

先给出不同 PTM 类型在 SS 与 Exposure 上的富集（OR、FDR）， 再对比不同组别/亚型的差异（对齐 [R1] 通过分组解释 PTM 主题的写法）。

Figure set B: Crosstalk (±5 aa → 3D clumps)

用 ±5 aa 共现筛候选串扰对 → 再做 3D clumps 验证 → 用 ASA/SASA 说明“位点可及性/机制合理性”。 这一步能自然衔接到“变异破坏 → 功能失衡 → 潜在治疗”。

Figure set C: Variant-driven PTM loss

把变异影响分为 direct / accessibility / interface/cluster， 并给出每类的代表例子（结构图 + ASA/SASA/SS 注释）。

Figure set D: Actionability

把 “差异位点集合/串扰对/3D clump/变异证据” 映射到上游酶与药物候选，形成可执行的靶点优先级清单。
